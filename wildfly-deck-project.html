WildFly Decks provides presentation materials for promoting WildFly on Conferences, JUGs, etc.

The Project

  • Sources: wildfly-extras/wildfly-decks at GitHub
  • Issue Tracker: wildfly-extras/wildfly-decks at GitHub
  • Project Leads: Eduardo (framework), Flavia (team coordination)

The Decks: Content

EAP Highlights, WildFly to EAP (reversed), Presenter Info, Cover, Cloud, Features (e.g Jakarta EE), Tooling, Community

The Decks: 2 Flavours of 45 min slides

  • WildFly Full Deck full focus on WildFly promotion as a project. Details all features, cloud, tooling, community.
  • WildFly Short Deck minimal presentation about WildFly as a project, a starting point for a component/topic specific extension, e.g. Undertow, or a live demo

The Framework

A fork of Dzslides, a single file html presentation framework.

Familiar team development process

  • Each deck is a html file (wildfly-deck-*.html)
  • You fork and customize your deck (at least the presenter info slide)
  • Slides are updated on upstream wildfly-extras/wildfly-decks repo, and you update your fork by rebasing upstream

All slides features you probably need...

  • Slides navigation using keyboard
  • Presentation progress bar at bottom
  • Styles for content animation
  • Slide Notes only shown on presenter interface
  • This presentation is actually being done with the framework :-)
